This is a recreation of my [university project](https://github.com/JonathanMSifleet/GameReviewWebsite)

*******************
To run:

1. Open terminal in project root
2. Enter "npm install"
3. Enter "npm start"
4. Open second terminal in roject root
5. Enter "ng serve"
